# enciclopedia-dashboard-data

Publishing endpoint for Dashboard data.json used by Enciclopedia iOS apps hybrid fetch (T6-S35).

## Source
Generated by `Dashboard/update_data.py` (launchd 6×/dia) in the main Enciclopedia repository.

## Endpoint
- Raw: `https://raw.githubusercontent.com/RicardoLSantos/enciclopedia-dashboard-data/main/data.json`
- Pages: `https://ricardolsantos.github.io/enciclopedia-dashboard-data/data.json`

## Used by
- 72H-Emergency · BlackoutReady · FireReady · HeatWave · StormReady · WaterCalculator

Hybrid fetch model: apps try remote first (timeout 2s), fall back to bundled dashboard.html if unreachable.

## Data scope
Public fuel/weather/seismic/financial/epidemiological aggregates from public APIs (PortWatch IMF, OilPrice RSS, Eurostat, USGS, WHO, ECDC, NASA EONET, GDACS, etc.). No personally identifiable information.

## Schema
Top-level keys (~30+): crypto, currencies, weather, financial, seismic, solar, health, commodities, fuel, basket, food_prices, historical, worldbank, volcanoes, air_quality, construction, gdacs, eonet_wildfires, portwatch_chokepoints, oilprice_rss, eurostat_oil_stocks, openfda_shortages, usgs_water, gdelt_events, epidemiological, thermometer, resilience, themes, coverage_map, source_uptime, meta.
